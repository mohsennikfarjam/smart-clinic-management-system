# User Stories - Smart Clinic Management System

## Admin User Stories
- **As an Admin**, I want to **log in to the portal**, so that I can **manage clinic resources and staff**.
- **As an Admin**, I want to **add a new doctor to the system**, so that they can **start accepting patient appointments**.
- **As an Admin**, I want to **view and manage doctor schedules**, so that the **clinic operations remain efficient**.

## Doctor User Stories
- **As a Doctor**, I want to **log in to my personal portal**, so that I can **access my schedule and patient information**.
- **As a Doctor**, I want to **view my daily list of patient appointments**, so that I can **prepare for each consultation**.
- **As a Doctor**, I want to **set and update my availability or time slots**, so that **patients can book appointments accurately**.
- **As a Doctor**, I want to **issue digital prescriptions to patients**, so that they can **receive their medication information promptly**.

## Patient User Stories
- **As a Patient**, I want to **log in to the patient portal**, so that I can **manage my healthcare interactions**.
- **As a Patient**, I want to **search for doctors by name or specialty**, so that I can **find the right medical professional for my needs**.
- **As a Patient**, I want to **view available time slots for a specific doctor**, so that I can **choose a convenient time for my appointment**.
- **As a Patient**, I want to **book an appointment online**, so that I can **secure my consultation without calling the clinic**.
- **As a Patient**, I want to **view my past and upcoming appointments**, so that I can **keep track of my medical history**.
